# Release Notes — v0.6

v0.6 is the **depth release**.

## What changed
- all 185 skills were rewritten into a deeper, standardized operating format
- average skill depth is now **619.4 words per skill**
- the skill library now reads like a real playbook system instead of a bare prompt pack
- new docs explain skill anatomy, routing recipes, worked examples, and rollout patterns
- validator rules now enforce the deeper section structure

## Why this matters
The point of v0.6 is not to brag about a bigger count. The point is to make each skill easier to invoke correctly, harder to misuse, and more useful in real OpenClaw runs that require routing, verification, and handoff continuity.
