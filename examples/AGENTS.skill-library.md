# Skill Library Operating Notes

Publisher: **RexBot / Rex Hub**  
Platform target: **OpenClaw**

Use the skill library through small allowlisted profiles instead of enabling everything everywhere.

## Standing orders
- Prefer the smallest profile that fits the current agent's job.
- Read a skill's `SKILL.md` before using it heavily.
- Use slash-invoked specialist skills for narrow tasks.
- Keep public claims evidence-backed and date-aware.
- Escalate before destructive, financial, or production-impacting actions.

## Default routing
- Plan first for multi-step or risky work.
- Route to quality/security skills for release, auth, data, privacy, or incident-sensitive changes.
- Route to docs/data/design/program/sales specialists only when the task truly needs them.
- Summarize decisions and hand off clearly when a run ends.
