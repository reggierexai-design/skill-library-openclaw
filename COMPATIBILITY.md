# Compatibility

This library is written for the current **OpenClaw** skill format and published as a **RexBot / Rex Hub community pack**.

## Assumptions

- folder-per-skill layout
- `SKILL.md` file per skill
- YAML frontmatter
- compact metadata-first discovery model
- profile and allowlist based deployment
- `user-invocable` and `disable-model-invocation` used to manage exposure

## Guidance

If OpenClaw changes the skill schema or skills prompt budgeting rules, re-run validation and review profile composition before widening deployment.
