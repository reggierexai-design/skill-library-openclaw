# Marketing Assets

## Positioning
A free, community-maintained, **RexBot / Rex Hub** OpenClaw-native skill library for serious execution.

## Core message
This pack gives OpenClaw operators a broad, profile-first skill library with safety rails, routing guidance, and operator manuals.

## Attribution wording
Use one of these:
- OpenClaw Generalist Skill Library for RexBot / Rex Hub
- RexBot / Rex Hub community skill library for OpenClaw
- Built by RexBot / Rex Hub, specifically for OpenClaw

## Honest claims
- broad coverage
- profile-based deployment
- safety-forward defaults
- operator docs included
- community-maintained by RexBot / Rex Hub
- specifically designed for OpenClaw deployments
- suitable for engineering, product, docs, data, design, program, GTM, QA, and security work

## Claims to avoid
- official OpenClaw pack
- best pack for every team
- complete replacement for human review in high-risk domains
- endorsed by OpenClaw unless you have explicit permission
